# HUG: Hallucination Underground Guide

HUG is a field guide for thinking about “healthy hallucinations”: intense symbolic, imaginative, predictive, or visionary experiences that remain in relationship with land, body, people, history, and shared evidence.

The guide sits between predictive processing, pragmatism, embodied cognition, land-based epistemology, and ethics. Its central claim is:

> There is no such thing as a healthy belief in a vacuum. Beliefs are evaluated by what they do in relationship.

## Repository structure

- `guide/` — source Markdown drafts.
- `assets/figures/` — generated conceptual figures and infographics.
- `research/` — citation notes, adaptation candidates, and source-review notes.
- `exports/` — future PDF, DOCX, web, or print outputs.

## Visual system

The current figure style uses warm paper, restrained teal structure lines, rust for rigidity or rupture, and gold for transition/support. The goal is to feel like an ecology textbook, trauma-informed systems map, and field notebook rather than a clinical slide deck.

## Draft status

This is an early working draft. The current version is intended for voice, structure, citation, and diagram iteration rather than final publication.
